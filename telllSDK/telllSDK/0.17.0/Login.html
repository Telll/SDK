Class: Login

Login

new Login(t)

View Login, implements the login widget It has 3 states: - "loginWait" - The widget was shown - "LoginDone" - The user filled fields and clicked login button - "authOk" - User authenticated The view emits event with the same name of status when reached. It uses the mustache template login_template.mtjs
